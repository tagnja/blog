## [plantuml-generator](https://github.com/tagnja/plantuml-generator)

一个关于 [PlantUML](https://plantuml.com/) 的工具。项目通过爬取 Java API Doc 文档的网页去生成符合 PlantUML 语法的文本，然后利用 plantuml.jar 将文本生成 UML 图表。
